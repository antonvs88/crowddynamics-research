Here are instructions for using the simulation codes in the folder.
NOTE! The locations for the data should be changed in the code.

crowddynamics
This folder includes the codes for running simulations where pedestrians have different strategies, but the strategies are fixed.

buildscripts
Scripts for installing the conda environment and dependencies.

requirements.txt
Dependencies.

run_gui.py
Run simulation with GUI.

run_script.py
Run simulation without GUI.

run_script.sh
Script which calls "run_script.py" and runs multiple simulations.
